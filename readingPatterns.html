Reading Patterns

MR: magic ring stitch

Technique to begin crochet projects by which you create a stitch/loop with an adjustable center to it.

SL ST: slip stitch

Technique of creating or joining a seamless edge within the crochet project. Insert hook, yarn over and pull through both loops.

CH: chain stitch

Foundation of projects/beginning a project by simply creating loops with the hook by pulling the yarn through.

SC: single crochet

Basic stitch. Insert hook, pull up a loop, yarn over again and pull through both loops on the hook.

HDC: half double crochet

An in between of sc and dc. Yarn over, insert hook into stitch, yarn over, pull through a loop, yarn over and pull through all three loops.

DC: double crochet

Creates a taller stitch than the sc. Yarn over, insert hook into stitch, yarn over, pull up a loop, yarn over, pull through two loops, yarn over and pull through remaining two loops.

TR: treble crochet

Creates a taller stitch than the dc. Yarn over twice, insert hook, yarn over, pull up a loop, yarn over, pull through two loops, yarn over, pull through two loops, yarn over and pull through remaining two loops.

FO: fasten off

Cut yarn and pull tail through last loop to secure stitch. We do this when finishing a piece or swtiching colours.

YO: yarn over

Wrap yarn over hook; creating additional loop on hook. This is used in many stitches.

INC: increase

Add stitches to row by working two or more stitches into the same stitch. This allows expansion of the row for more stitches in the next.

DEC: decrease

Combining two or more stitches into one to reduce the stitch count This is done through looping two or more consecutive stitches onto the hook before yarning over.

BLO: back loop only

Working stitches into the back loop of the stitch only.

FLO: front loop only

Working stitches into the front loop of the stitch only.

ST: stitch

Refers to individual loops.

* : Repeat

Asterisk used to mark the beginning of a sequence that is to be repeated. For example (2sc)*3 is 2 single crochets to be repeated 3 times.

( ) : Group

Brackets used to distinguish a group of stitches to be worked for example in one stitch. Example: 2 sc, (2 hdc, 3 dc, 2 hdc) in next st.
